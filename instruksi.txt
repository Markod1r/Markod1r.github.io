HTML:
-- element --
<h_></h_>           = header -> judul, sub-judul
<p></p>             = paragraf -> kalimat
<br>                = line break -> tag html untuk memisahkan baris teks tanpa membuat paragraf baru
<img>               = image -> tampilkan gambar
<div></div>         = element container untuk mengelompokkan beberapa elemen untuk mempermudah styling
<section></section> = digunakan untuk pengelompokkan konten yang punya topik yang sama
<article></article> = untuk menandai konten yang dapat berdiri sendiri dan punya makna tersendiri (berita, blog, komentar)
<aside></aside>     = untuk menampilkan konten sampingan yang berkaitan dengan konten utama (sidebar, iklan, dll)

-- atribut --
class=""    = untuk memberikan nama pada elemen supaya mudah saat styling atau melakukan manipulasi dengan Javascript.
(nama elemen pada class bisa dipake lebih dari satu elemen)
id=""       = atribut unik untuk memberi identitas khusus pada satu elemen.
src=""      = menentukan sumber file eksternal (gambar, audio, video)
alt=""      = alternatif teks jika gambar tidak muncul

Styling:
color               = mengubah warna teks
background-color    = mengubah warna latar belakang pada elemen
width               = menentukan lebar suatu elemen 
height              = menentukan tinggi suatu elemen
padding             = ruang di dalam elemen yang memberi jarak antara konten dengan border   

nilai satuan:
px = pixel
rem = root em -> direkomendasikan kalau mau buat halaman responsif.


Semantik HTML:
menggunakan elemen HTML dengan arti dan makna jelas, untuk mempermudah browser, developer, dan mesin pencari
memahami struktur halaman web.
<header></header>   = bagian atas halaman (judul, logo, menu)
<main></main>       = main konten yang ingin ditampilkan pada halaman web
<footer></footer>   = bagian bawah atau penutup halaman (hak cipta, kontak)


TUTORIAL - BOLEH BACA YANG INI ATAU CARI DI INTERNET

Berikut tutorial membuat akun GitHub :
1. Membuat Akun GitHub:
    Buka situs web GitHub: https://github.com/.
    Klik "Sign up" di pojok kanan atas.
    Isi formulir pendaftaran dengan informasi yang diminta (nama, email, password).
    Verifikasi email Anda melalui tautan yang dikirimkan ke alamat email yang Anda daftarkan. 

2. Membuat Repositori (Repo) di GitHub:
    Setelah login, klik ikon "+" di pojok kanan atas, lalu pilih "New repository".
    Isi nama repositori, deskripsi (opsional), dan pilih apakah repositori akan bersifat publik atau privat.
    Klik "Create repository".
